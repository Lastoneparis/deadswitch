SECURITY MODEL — THREAT MITIGATIONS

Wrong person claims?
Multi-factor: pre-approved address (set at vault creation) + World ID (anti-bot) + optional secret phrase. Three layers.
Shards compromised?
3-of-5 Shamir threshold — attacker needs 3 shards from different storage locations. Encrypted at rest.
Owner forgets heartbeat?
Multi-channel alerts (email, push, on-chain event). 30-day warning period before recovery. Owner can resume anytime.
Heir loses access?
Owner can update beneficiary anytime while vault is active. Multiple heirs supported in future version.
Monetization is future work. Today's focus: build the infrastructure right.
